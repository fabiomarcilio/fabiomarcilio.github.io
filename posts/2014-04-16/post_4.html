O bootstrap é um conteudo que basicamente trata de varios links de CSS com algumas funções de Javascript embarcadas que te permitem fazer um website completo e bastante atual. Ele possui também diversos plugins de Javascript prontos para o uso que, com um simples CTRL+C e CTRL+V você poderá criar menus com dropdown, alertas, sliders, etc. Claro, a personalização fica a seu critério (os padrões disponibilizados pela ferramenta não são antiquados!). O objetivo principal e lógico do Bootstrap é consumir o menor tempo possível no desenvolvimento de um website, seja ele uma página simples estática ou um grande portal dinâmico. LayoutIt é um construtor de interfaces que utiliza o bootstrap para definir os estilos e elementos a serem utilizados. O seu funcionamento é muito simples, basta escolher o tipo de recurso que deseja, clicar, arrastar e soltar onde quiser. Você poderá definir o grid da página formatando todo o conteúdo como achar melhor. É possivel criar páginas com várias colunas de conteúdo e se o resultado final não for o desejado basta voltar, excluir e refazer o que for necessário. Depois de finalizado é só fazer o download dos arquivos. Se quiser o HTML é só copiar e utilizar em qualquer projeto.
